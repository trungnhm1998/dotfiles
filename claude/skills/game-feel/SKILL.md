# Game Feel

Make mechanics feel satisfying without drowning the player in noise. Present options with trade-offs, then recommend; name the lever being pulled so the user learns it.

## Use the references
- The core juice levers + Unity how-tos → `references/feel-checklist.md`.
- Layering and readability of feedback → `references/feedback-patterns.md`.
- Camera feel (Cinemachine, follow, shake, framing) → `references/camera-feel.md`.

## Principles
- Readable first, flashy second — every effect must communicate something.
- Tune with curves/easing, not linear lerps; shape anticipation → impact → follow-through.
- Tools: PrimeTween/DOTween (tweening), Feel/MMFeedbacks (prebuilt juice). Decide buy-vs-build via the indie-production rubric (the user leans build).
