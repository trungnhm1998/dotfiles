# Steam Page Guide

## When to Go Live

**Open your Steam page as early as possible** — ideally when you have:
- A capsule image (header + small capsule)
- 5+ screenshots that show actual gameplay
- A short description that conveys the game clearly
- A rough launch window (even "2025" is fine)

Every day the page is live is a day wishlists can accumulate. Going live 6–12 months before launch is common for successful indie games.

## Required Assets

| Asset | Size | Notes |
|---|---|---|
| **Header Capsule** | 920 × 430 px | Most visible; shown in search. Must read at small sizes. (Doubled from legacy 460×215 in Aug 2024.) |
| **Small Capsule** | 462 × 174 px | Used in lists, "New on Steam" widgets |
| **Main Capsule** | 1232 × 706 px | Featured/Recommended section |
| **Library Capsule** | 600 × 900 px | Player's library view |
| **Screenshots** | Min. 5; 1280 × 720 or higher | First screenshot is shown as thumbnail. Gameplay > art > key art. |
| **Trailer** | Highly recommended | Autoplays on page. First 30s critical. |

**Capsule art principles:**
- Legible at thumbnail size (the small capsule is tiny).
- Character or key visual — faces attract attention.
- Title readable even in small capsule.
- Consistent style with the game's art direction.
- Avoid excessive text (the title is enough).

## Short Description

~300 characters max (Steamworks enforces the limit on submission). Appears in search results and lists. Must answer: **what do you do? what's the hook?**

Bad: "An action RPG with an epic story and hundreds of items."  
Good: "A one-button rhythm platformer where every obstacle pulses to the beat."

Formula: [core mechanic] + [unique hook] + [optional tone].

## Long Description

Appears on the store page below the media. Structure:

1. **Hook sentence** — the game's elevator pitch.
2. **Features bullet list** — 4–6 bullets, each concrete: "100 hand-crafted levels", "permadeath with persistent upgrades", "original 8-bit OST".
3. **Description paragraph** — expand on the game world/tone; 1–2 short paragraphs.
4. **More details / credits** — optional.

Avoid: vague superlatives ("incredible", "unique"), long plot synopsis (players care about what they *do*, not lore), walls of text.

## Tags (critical for discovery)

Steam tags drive algorithmic discovery ("More Like This"). Choose them carefully:

1. In Steamworks → Store Page → Tags: add up to 20.
2. **Most important**: the genre tags that match your game's core loop (Roguelike, Platformer, RPG, etc.).
3. **Secondary**: audience/theme tags (Pixel Art, Cute, Dark, Multiplayer, Singleplayer, etc.).
4. **Research**: look at top-selling similar games and use their tags where accurate.

Poor tagging = the Steam algorithm can't place you next to relevant games = fewer wishlists from organic discovery.

## System Requirements

Fill these in honestly. Default to the widest compatibility you've verified. Err conservative on minimums.

## Before Going Live Checklist

- [ ] Capsule images uploaded (all sizes)
- [ ] 5+ gameplay screenshots
- [ ] Short description finalized
- [ ] Tags selected (15–20 relevant)
- [ ] Mature content survey completed
- [ ] Languages set
- [ ] System requirements filled
- [ ] Upcoming release date or "Coming Soon" set
- [ ] Trailer uploaded (or schedule soon after launch)
